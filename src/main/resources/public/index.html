Welcome to Nubank invitation challenge service.

Available operations:
HTTP Method Resource URI Payload Content-Type Response Brief explanation
GET /ranking JSON Array A list containing users' scores
POST /invitation { "invitee": Integer, "inviter": Integer } "application/json" String message Inserts a new invitation in the InvitationTree. This request will be ignored if the inviter does not exist.
